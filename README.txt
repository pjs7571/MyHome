CircuitAI Studio - AI Smart Builder Final

실행 방법:
1. 압축을 푼다.
2. index.html을 Chrome 또는 Edge로 연다.
3. 오른쪽 AI 도우미에 원하는 회로를 문장으로 입력한다.

예시 명령:
- AND 회로 LED 4개 직렬로 달아서 작동되게 해줘
- AND 회로 오실로스코프로 측정해줘
- OR 회로 LED 2개 만들어줘
- NOT 회로 멀티미터로 측정해줘
- 아두이노 13번 LED 3개 켜줘
- DC 모터 회로 만들어줘

추가 내용:
- 버튼 ON 시 눌린 모양 표시
- LED 발광 효과
- 오실로스코프/멀티미터 자동 연결
- LED 개수 자동 인식
- 논리회로 AND/OR/NOT/NAND/NOR/XOR/XNOR 대응
- 브레드보드 정렬 및 선 연결 안정화
